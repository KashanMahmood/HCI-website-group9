P5 : Final Prototype

Implementing a functional prototype using a programming platform.

Process

For this milestone, we used results from the heuristic evaluation we received to revise and improved our interactive prototype. Once we fixed the critical design violations on our interactive prototype, we started building a functional prototype using Flutter. We build the user flows for the hard and easy tasks outlined earlier, and in the interest of time, we did not implement the medium task and few extra pages. We summarised this process and deliverables in a report and also recorded a demo video of our funtioning prototype on Kashan's iPhone!
